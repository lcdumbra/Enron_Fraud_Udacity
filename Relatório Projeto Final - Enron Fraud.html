Validação¶

Valida√ß√£o de modelo √© definida como o processo em que um modelo treinado √© avaliado com um conjunto de dados de teste.
O conjunto de dados de teste √© uma parte separada do mesmo conjunto de dados do qual o conjunto de treinamento √© derivado.
O principal objetivo do uso do conjunto de dados de teste √© testar a capacidade de generaliza√ß√£o de um modelo treinado.
A valida√ß√£o do modelo √© realizada ap√≥s o treinamento do modelo.

Treinar o modelo de previs√£o e test√°-lo no mesmo conjunto de dados usado para teste √© um erro cl√°ssico.
Um modelo que apenas repetiria os r√≥tulos das amostras que acabou de ver teria uma pontua√ß√£o perfeita, mas n√£o poderia prever nada de √∫til em dados ainda n√£o vistos. Isso √© chamado 'overfitting'.
Para evit√°-lo √© comum ao realizar um experimento de aprendizado de m√°quina (supervisionado), reter parte dos dados dispon√≠veis como um conjunto de testes.
